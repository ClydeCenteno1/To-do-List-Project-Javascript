# 📝 Todo List App

A simple browser-based Todo List app built with vanilla HTML and JavaScript.

## Features

- **Add todos** – add as many items as you want
- **List todos** – view all current todos with their index
- **Delete todos** – remove a specific todo by index
- **Quit** – exit the app

## How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/todo-app.git
   ```
2. Open `index.html` in your browser.
3. Follow the prompts in the browser dialog boxes.

## Commands

| Command  | Description              |
|----------|--------------------------|
| `new`    | Add a new todo item      |
| `list`   | List all todos           |
| `delete` | Remove a todo by index   |
| `quit`   | Exit the app             |

## Tech Stack

- HTML5
- Vanilla JavaScript
